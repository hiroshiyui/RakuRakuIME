# Phone screenshots (en-US)

Drop 1080-wide (or similar phone-aspect) PNG/JPEG screenshots in this
folder. Both F-Droid and Google Play read them from here directly;
Google Play additionally allows 16:9 landscape screenshots.

**Filename convention:** `1_<slug>.png`, `2_<slug>.png`, … — the leading
number controls ordering in stores.

**Required minimum:** 2 screenshots for Google Play, though 4–8 is
conventional so the screenshot carousel has weight.

**Suggested coverage for this app:**
1. Settings screen showing the IME Status row and the Theme selector.
2. EZ (輕鬆輸入法) keyboard with a live candidate bar mid-composition.
3. English keyboard with word-prediction candidates.
4. Emoji panel with the category tabs visible.

No images live here yet — capture them from a real device or emulator
once the UI settles.
