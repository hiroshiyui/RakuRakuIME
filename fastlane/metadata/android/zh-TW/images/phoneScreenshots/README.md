# 手機螢幕截圖（zh-TW）

在此資料夾放入手機比例的 PNG / JPEG 截圖。F-Droid 與 Google Play 都會直接
從這裡讀取；Google Play 也接受 16:9 的橫向截圖。

**檔名慣例：** `1_<說明>.png`、`2_<說明>.png`…… 開頭的編號決定商店頁面上
的排列順序。

**最少需求：** Google Play 至少需要 2 張截圖，但商店頁面建議上架 4–8 張。

**建議內容：**
1. 設定畫面（顯示輸入法狀態列與佈景主題選擇）。
2. 輕鬆輸入法鍵盤，含即時候選字列。
3. 英文鍵盤展示單字預測。
4. 表情符號面板（可見分類標籤）。

尚未放入任何圖片，請於 UI 穩定後自實機或模擬器擷取。
